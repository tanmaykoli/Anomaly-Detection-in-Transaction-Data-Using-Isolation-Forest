# Anomaly Detection in Transaction Data Using Isolation Forest

## Project Overview
This project detects abnormal or potentially fraudulent transactions using the Isolation Forest algorithm. The model analyzes transaction patterns such as amount and time to identify unusual behavior without requiring labeled data.

## Objective
- Understand anomaly detection
- Detect suspicious transactions
- Apply unsupervised machine learning
- Visualize anomalies

## Dataset
Synthetic transaction dataset containing:
- Transaction ID
- Amount
- Time
- Merchant
- Location

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Methodology
1. Generate transaction dataset
2. Perform exploratory analysis
3. Train Isolation Forest model
4. Detect anomalies
5. Visualize abnormal transactions
6. Save trained model

## Results
- Successfully detected abnormal transactions
- Visualized anomalies clearly
- Demonstrated fraud detection workflow
